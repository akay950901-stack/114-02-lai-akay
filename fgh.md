# 地鼠大冒險 (Mole Adventure)

這是一款使用 React + TypeScript 開發的卡通風格休閒打地鼠遊戲。

## 遊戲特色
- **豐富的地鼠種類**:
  - 棕色地鼠: 基礎分 1 分。
  - 黃色地鼠: 速度較快，得分 3 分。
  - 紅色地鼠: 極速出現，得分 5 分。
- **強力道具**:
  - **地鼠錘 (Hammer)**: 5 秒內得分翻倍 (x2)。
  - **地鼠網 (Net)**: 凍結所有地鼠 3 秒。
- **關卡系統**: 共有 5 個關卡，難度隨關卡提升（地鼠生成頻率加快、停留時間縮短）。
- **生命機制**: 初始 5 顆心，漏掉地鼠會扣血。
- **視覺與音效**:
  - 全 SVG 手繪組件，支持任意解析度。
  - 內建 Web Audio API 合成音效，無需額外資源。

## 如何運行
1. 確保已安裝 Node.js。
2. 在項目根目錄執行 `npm install`。
3. 執行 `npm run dev` 開啟開發伺服器。
4. 在瀏覽器中打開生成的鏈接即可開始遊戲。
